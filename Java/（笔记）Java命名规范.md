## 2. Some Notes

* 命名有意义一些，查一下词典
* 下标一般用 `index`，或者简拼 `idx`
* 临时变量：`tmp`
* 少用 `i`、`j`，用更有含义、容易明白些的头指针 `front`、尾指针 `rear`之类的
